PDF 분할 프로그램

하나의 PDF 파일을 원하는 페이지 수 단위로 나눠 ZIP 파일로 저장해 드립니다. 모든 처리는 이 브라우저 안에서만 이루어지고, 파일이 외부로 전송되지 않습니다.

STEP 01
PDF 선택
분할할 PDF 파일 1개를 클릭하거나 끌어다 놓으세요. 업로드하면 총 페이지 수를 바로 확인할 수 있습니다.
STEP 02
분할 설정
몇 페이지씩 나눌지 입력하고, 파일명 형식을 예시(예: PDF파일_001)처럼 입력하면 끝 번호 자리수를 자동으로 인식해 순서대로 번호를 매깁니다.
STEP 03
분할 & 저장
[분할하기]를 누르면 설정한 페이지 수 단위로 잘라 ZIP 파일 하나로 묶어 드립니다. 압축을 풀면 개별 PDF 파일들이 나옵니다.
파일명 형식 규칙. 예시의 맨 끝 숫자 자리수를 그대로 사용합니다. "PDF파일_001"이면 3자리(001, 002, …), 숫자가 없으면 자동으로 "_001" 형식이 붙습니다. 분할된 파일 수가 자리수보다 많으면 자릿수가 자동으로 늘어납니다.

대용량 PDF 팁. 페이지 수가 많은 파일도 처리할 수 있지만 용량이 클수록 시간이 걸릴 수 있어요. 처리 중에는 브라우저 탭을 닫지 마세요.

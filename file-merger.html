파일 통합 프로그램

여러 개의 엑셀 파일 혹은 여러 개의 PDF 파일을 한 번에 올리면, 엑셀은 모든 시트를 하나의 통합 워크북으로, PDF는 모든 페이지를 하나의 PDF로 순서대로 합쳐 드립니다. 모든 처리는 이 브라우저 안에서만 이루어지고, 파일이 외부로 전송되지 않습니다.

STEP 01
파일 선택
합칠 엑셀 파일들 또는 PDF 파일들을 클릭하거나 끌어다 놓아 한 번에 여러 개 추가하세요. 형식은 자동으로 인식되며, 엑셀과 PDF는 함께 합칠 수 없어 다른 형식을 올리면 안내와 함께 제외됩니다. 같은 파일을 여러 번 추가해도 중복 없이 한 번만 처리됩니다. 목록에서 각 항목 왼쪽의 ⠿ 를 드래그하면 순서를 바꿀 수 있습니다.
STEP 02
합치기
[합치기]를 누르면 파일을 업로드된 순서대로 읽어 하나로 합칩니다. 엑셀은 시트 단위로, PDF는 페이지 단위로 순서를 그대로 유지합니다. 손상된 파일은 자동으로 건너뛰고 목록에 표시됩니다.
STEP 03
저장
파일명을 확인하거나 수정한 뒤 저장하세요. Chrome/Edge 최신 버전에서는 저장 위치도 직접 선택할 수 있습니다.
엑셀 시트 이름 규칙. 파일에 시트가 1개면 "파일이름"으로, 여러 개면 "파일이름_시트이름"으로 자동 지정됩니다. 이름이 31자를 넘거나 다른 시트와 겹치면 자동으로 조정됩니다.

PDF 병합 방식. 각 PDF의 모든 페이지가 업로드한 순서대로, 원본 페이지 순서를 유지한 채 그대로 이어붙습니다.

대량 처리 팁. 100개 이상의 파일도 처리할 수 있지만, 파일 수와 용량이 클수록 시간이 걸릴 수 있어요. 처리 중에는 브라우저 탭을 닫지 마세요.
